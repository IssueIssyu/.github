# 📢 이슈있슈 (IssueIssyu)

> 
> **우리 동네 맞춤형 이슈 리포팅 및 로컬 정보 지도형 커뮤니티 서비스** 
> 
> 

이슈있슈(IssueIssyu)는 복잡한 민원 신고 절차를 간소화하고 시민들의 자발적인 문제 해결 참여를 독려하는 하이퍼로컬 시빅테크(Civic Tech) 플랫폼입니다. 단순히 문제를 신고하는 것에 그치지 않고, 지도 기반으로 동네 문제를 직관적으로 공유하며 이웃과 공감하고, 실제 해결 과정을 투명하게 확인할 수 있는 새로운 커뮤니티 환경을 제공합니다.

## ✨ Core Features

**1. 지도 기반 리포팅 (Map & Reporting)**

* **동네 이슈 공유**: 길을 걷다 발견한 불편함부터 소소한 축제까지 이슈, 이벤트, 소통 핀을 지도에 꽂아 이웃들과 빠르게 상황 공유
* **진척도 및 패치노트**: 내가 올린 이슈의 해결 상황(진행 전/중/완료)을 실시간으로 확인하고, 완료된 이슈는 '패치노트'에 차곡차곡 기록되어 동네가 발전하는 모습을 한눈에 파악

**2. 커뮤니티 및 소셜 참여 (Community & Action)**

* **커뮤니티**: 지도에 등록된 핀 중 주민들의 폭풍 공감(좋아요)을 받은 핀을 커뮤니티로 자동 이동시켜, 더 많은 사람과 동네 주요 관심사에 대해 깊이 있는 소통 유도
* **시민해결사 & 지자체 SOS**: 사용자가 직접 나서는 '시민해결사' 기능과 혼자 해결하기 버거운 문제에 대해 지자체에 도움을 요청하는 '청원' 연계 시스템을 통해 확실한 문제 해결 지원

**3. 로컬 정보 및 알림 (Local Info & Notification)**

* **정책 카드뉴스**: 놓치기 아까운 동네 정책과 공모전 소식을 커뮤니티에서 제공하며, 복잡한 내용은 카드뉴스 형태로 변환하여 누구나 쉽고 재미있게 정보 습득 가능
* **실시간 소식 알림**: 우리 동네에서 일어나는 HOT 이슈와 중요 소식들을 맞춤형 알림으로 가장 먼저 전달
